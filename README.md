# WebFramework

A web application built with Vue 3 and Express.

## Technologies Used

- **Frontend:** 
  - Vue 3.5.13
  - Vue Router 4.5.0
  - Pinia 3.0.2
  - Axios 1.8.4

- **Backend:**
  - Express 5.1.0
  - MongoDB 6.16.0
  - Nodemon 3.1.9

- **Build Tools:**
  - Vite 6.2.4
  - @vitejs/plugin-vue 5.2.3
  - vite-plugin-vue-devtools 7.7.2

## Installation

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
